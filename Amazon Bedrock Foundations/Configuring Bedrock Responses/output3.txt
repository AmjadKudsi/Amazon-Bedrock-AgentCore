# AWS Lambda Overview

AWS Lambda is a **serverless compute service** that lets you run code without provisioning or managing servers. You simply upload your code, and Lambda automatically handles the compute resources needed to run it.

## How Lambda Works

1. **Upload your code** as a Lambda function
2. **Set up triggers** from AWS services, HTTP endpoints, or in-app activity
3. **Lambda automatically runs** your code when triggered
4. **Pay only** for the compute time you consume
